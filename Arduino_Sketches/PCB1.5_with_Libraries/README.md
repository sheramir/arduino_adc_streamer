# PCB1.5_with_Libraries

This folder contains a new modular split of the PCB1.5 firmware into reusable libraries and new main sketches.

## Layout
- `Teensy/` - Modular Teensy master sketch + reusable libraries.
- `MG24/` - Modular MG24 sketch + reusable libraries.

No existing sketches in other folders were modified.
