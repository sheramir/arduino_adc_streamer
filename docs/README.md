# docs

Documentation for the Arduino ADC streamer project, split into current-behavior references, user guides, and historical/planning records.

## Subfolders

- `architecture/` — implementation notes for active (non-legacy) subsystems: how a feature is currently organized across the codebase. See `architecture/README.md`.
- `user/` — user-facing guides for configuring and using current application features (array sensor configuration, heatmap tab). See `user/README.md`.
- `history/` — historical milestone completion notes and forward-looking refactor/roadmap plans, including the `refactoring_log/` subfolder of phase-by-phase modularization reports. See `history/README.md`.
