# user

User-facing guides for configuring and using current application features.

## Files

- `ARRAY_CONFIGURATION_GUIDE.md` — guide to the `array_layout` sensor configuration type: the 3x3 array grid model, sensor ID format, MUX/channel assignment, where configurations are stored, how to create/edit a layout in the Sensor tab, an example JSON shape, validation rules, and a troubleshooting table.
- `HEATMAP_README.md` — guide to the Heatmap tab: what it renders (live center-of-pressure view from rolling capture buffers), the runtime file map for the active heatmap path, input requirements for channel-layout vs array-layout modes, the piezo/PZT vs 555/PZR processing modes, where last-used settings are persisted, available controls, and performance notes.
